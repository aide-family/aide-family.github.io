# 下载镜像

:::tip
由于一些众所周知的原因, 所以 docker hub 上的镜像可能需要自己解决一下环境
:::

## Palace服务
:::info
 [palace docker 仓库地址](https://hub.docker.com/r/aidemoonio/palace/tags)
:::

```bash title="下载palace镜像"
docker pull aidemoonio/palace:latest
```
